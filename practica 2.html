Bienvenido a mi pagina HTML

Html es el lenguaje con el que se escriben las pagginas web. las paginas web pueeden ser vistas por el usuario mediante un tipo de aplicacion llamada navegador

Este lenguaje nos permite aglutinar textos, sonidos imagenes y combinarlos a nuestro gusto.
